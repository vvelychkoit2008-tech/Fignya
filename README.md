## 🎨 Про проект

**Fihnya** — це повноцінний UI/UX інструмент для дизайну та прототипування, що працює прямо у вашому браузері. Він натхненний такими індустріальними стандартами як Figma, проте зберігає максимальну легкість та швидкодію, оскільки не використовує важких залежностей.

Від створення простих векторних ілюстрацій до розробки складних інтерактивних прототипів з Auto Layout — **Fihnya** має все необхідне.

---

## 🚀 Основні Можливості

### 📐 Інструменти малювання
* **Примітиви:** Прямокутники, Еліпси, Трикутники, Зірки.
* **Текст:** Повна підтримка типографіки (шрифти, товщина, розмір).
* **Перо (Path Tool):** Створення складних векторних кривих (з двоетапною моделлю малювання).
* **Зображення:** Імпорт та масштабування растрових зображень.
* **Фрейми та Групи:** Об'єднання об'єктів для зручної роботи.

### ⚡ Професійні UI фічі
* **Auto Layout:** Автоматичне вирівнювання (Flexbox-стиль) всередині фреймів — підтримка відступів (Padding), проміжків (Gap) та напрямків (Vertical/Horizontal).
* **Прототипування:** Режим **Prototype**, який дозволяє "зв'язувати" об'єкти з фреймами для створення інтерактивних клікабельних презентацій.
* **Панель шарів:** Деревовидна структура шарів з підтримкою перетягування (Drag-and-Drop), реорганізації, приховування та блокування об'єктів. Масове виділення з підтримкою `Ctrl` та `Shift`.
* **Панель Властивостей:** Повний контроль над координатами, розмірами, радіусом кутів, прозорістю, кольорами заливки (Fill) та контуру (Stroke).

### ☁️ Синхронізація та Збереження
* **Хмарне сховище:** Інтеграція з Google Drive API дозволяє миттєво зберігати ваші проекти в хмару та завантажувати їх звідти.
* **Локальне сховище:** Якщо інтернет відсутній, усі зміни зберігаються у `localStorage` браузера.
* **Експорт:** Можливість експортувати проект у форматах `.png`, `.svg` та `.json` (для міграції проектів).

## 🛠 Технологічний Стек

* **Core:** Vanilla JavaScript (ES6+). Архітектура ООП.
* **Рендеринг:** SVG DOM для малювання графічних об'єктів (забезпечує ідеальну якість при будь-якому зумі) та HTML5 для інтерфейсу користувача.
* **Стилізація:** Vanilla CSS (CSS Variables, Flexbox, Grid, кастомні скролбари, Dark Theme).
* **Іконки:** [Lucide Icons](https://lucide.dev/).
* **Хмара:** Google Identity Services (GIS) та Google Drive API v3.

## 🏎 Встановлення та Запуск

Оскільки проект написаний повністю на клієнтських технологіях, не потрібно жодного процесу збірки (No Webpack / No Vite). 

Проте, для правильної роботи **Google OAuth** та браузерних політик (CORS) проект має бути запущений через локальний веб-сервер.

1. **Склонуйте репозиторій або завантажте файли.**
2. **Відкрийте теку в терміналі.**
3. **Запустіть локальний сервер.** Наприклад, за допомогою Node.js:
   ```bash
   npx serve .
   ```
   *Або через Python:*
   ```bash
   python -m http.server 3000
   ```
4. Відкрийте браузер за адресою: `http://localhost:3000`

> **Важливо:** Google Auth не працюватиме, якщо ви просто відкриєте `index.html` як локальний файл (`file:///...`).
